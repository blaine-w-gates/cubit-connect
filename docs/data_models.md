# Data Models: Cubit Connect

## Core Interfaces

### TaskItem
Represents a single task extracted from the video transcript.

```typescript
interface TaskItem {
  id: string;                    // UUID via crypto.randomUUID()
  task_name: string;             // Short task title from AI
  timestamp_seconds: number;     // Position in video (seconds)
  description: string;           // Detailed description from AI
  screenshot_base64: string;     // JPEG data URI, max 640px width
  sub_steps?: string[];          // Optional: Generated by Cubit feature
}
```

**Storage:** IndexedDB via `idb-keyval`

---

### AppState (Zustand Store)
Global application state.

```typescript
interface AppState {
  // Data
  apiKey: string | null;
  videoFile: File | null;        // RAM only, not persisted
  transcript: string | null;
  tasks: TaskItem[];
  
  // UI State
  isProcessing: boolean;
  processingProgress: number;    // 0-100
  processingStatus: string;      // "Parsing transcript...", "Capturing frame 3/10..."
  error: string | null;
  
  // Flags
  isPrivateBrowsing: boolean;    // True if IndexedDB unavailable
  needsVideoRehydration: boolean; // True if tasks exist but video missing
  
  // Actions
  setApiKey: (key: string) => void;
  clearApiKey: () => void;
  setVideoFile: (file: File) => void;
  setTranscript: (text: string) => void;
  setTasks: (tasks: TaskItem[]) => void;
  updateTask: (id: string, updates: Partial<TaskItem>) => void;
  setProcessing: (status: boolean, progress?: number, message?: string) => void;
  setError: (error: string | null) => void;
  reset: () => void;
}
```

**Storage:** Zustand with custom persistence middleware

---

### ProcessingQueueItem
Tracks async screenshot capture progress.

```typescript
interface ProcessingQueueItem {
  taskId: string;
  timestamp: number;
  status: 'pending' | 'processing' | 'completed' | 'error';
  retryCount: number;
}
```

**Storage:** In-memory only (not persisted)

---

### AIRequestQueueItem
Rate-limited queue for Gemini API calls.

```typescript
interface AIRequestQueueItem {
  id: string;
  type: 'analyze' | 'cubit';
  payload: string;               // Transcript text or context
  status: 'pending' | 'processing' | 'completed' | 'error';
  result?: TaskItem[] | string[];
  error?: string;
}
```

**Storage:** In-memory only

## Storage Strategy

### LocalStorage Keys

| Key | Type | Description |
|-----|------|-------------|
| `cubit_api_key` | `string` | Gemini API key |
| `cubit_theme` | `'light' \| 'dark'` | User theme preference |

### IndexedDB Keys (via idb-keyval)

| Key | Type | Description |
|-----|------|-------------|
| `cubit_tasks` | `TaskItem[]` | All extracted tasks with screenshots |
| `cubit_transcript` | `string` | Raw transcript text |

## Validation Rules

### API Key
- Must start with `AI` (Gemini keys format)
- Must pass `models.list()` test call
- Stored only after successful validation

### VTT/SRT Input
- Sanitized via DOMPurify before parsing
- Loose regex tolerates non-standard headers
- Empty result = show "No captions found" error

### Screenshot Base64
- Must be valid JPEG data URI
- Max dimensions: 640px width
- Quality: 0.7 compression

## Type Guards

```typescript
function isTaskItem(obj: unknown): obj is TaskItem {
  return (
    typeof obj === 'object' &&
    obj !== null &&
    typeof (obj as TaskItem).id === 'string' &&
    typeof (obj as TaskItem).task_name === 'string' &&
    typeof (obj as TaskItem).timestamp_seconds === 'number'
  );
}

function isTaskArray(arr: unknown): arr is TaskItem[] {
  return Array.isArray(arr) && arr.every(isTaskItem);
}
```
